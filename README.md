## Proyectos en Producción

### FreeFlow CRM — SaaS Freemium "Lead-to-Cash"
> CRM para freelancers que automatiza el ciclo completo: Cliente → Propuesta → Proyecto → Factura → Cobro

- Monetización real con **Stripe Checkout + Billing Portal** (plan gratuito / Pro a $19/mes)
- Suscripciones recurrentes via webhooks (`checkout.session.completed`, `subscription.deleted`)
- Autenticación SSR con Row Level Security — cada usuario accede solo a sus datos
- Stack: `Next.js 16` · `React 19` · `TypeScript 5` · `Supabase` · `Stripe` · `Tailwind 4` · `Radix UI`

### DocuMente — Motor RAG de Alta Precisión
> Análisis documental con IA para el mercado contable y legal colombiano

- Búsqueda semántica con latencia de milisegundos usando **Vector Search nativo en PostgreSQL**
- Orquestación híbrida **Groq + Gemini** para balancear velocidad vs. análisis profundo
- Stack: `Next.js 15` · `React 19` · `TypeScript` · `Supabase` · `PostgreSQL pgvector` · `Groq` · `Gemini`
