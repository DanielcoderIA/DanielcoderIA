### `[01] ⚡ FREEFLOW CRM — SaaS Freemium "Lead-to-Cash"`

<img src="https://img.shields.io/badge/●_STATUS-LIVE-CC0000?style=flat-square&labelColor=000000" />
<img src="https://img.shields.io/badge/PLAN_PRO-$19%2Fmes-333333?style=flat-square&labelColor=111111" />

<img src="https://img.shields.io/badge/NEXT.JS-15-ffffff?style=flat-square&logo=nextdotjs&logoColor=000000&labelColor=ffffff" />

**Plataforma de gestión financiera integral para agencias y freelancers.**
Automatiza el ciclo comercial completo — desde la captación del lead hasta el cobro final.

```
FLUJO  →  Cliente  ▸  Propuesta  ▸  Proyecto  ▸  Factura  ▸  Cobro
```

**LOGRO CLAVE:** Ciclo de vida del cliente 100% centralizado. Facturación automatizada vía Stripe con monitoreo de KPIs financieros en tiempo real, eliminando la dependencia de herramientas externas costosas.

| MÓDULO | IMPLEMENTACIÓN TÉCNICA |
|---|---|
| 💳 **Monetización real** | Stripe Checkout + Billing Portal · Plan Free / Pro $19/mes · pagos activos en producción |
| 🔄 **Suscripciones recurrentes** | Webhooks `checkout.session.completed` · `customer.subscription.deleted` · manejo completo de estados |
| 🔐 **Autenticación multitenant SSR** | Row Level Security en Supabase — aislamiento total de datos por tenant, sin exposición cruzada |
| ⚡ **Performance arquitectural** | Server Components + Streaming — hidratación mínima del cliente, cero layout shift |
| 📊 **KPIs en tiempo real** | Dashboard live vía Supabase Realtime — actualizaciones push, sin polling ni cron jobs |

```
STACK: Next.js 15 · React 19 · TypeScript 5 · Supabase · PostgreSQL · Stripe · Tailwind CSS 4 · Radix UI
```

### `[02] 🧠 DOCUMENTE — Motor RAG de Alta Precisión`

<img src="https://img.shields.io/badge/●_STATUS-LIVE-CC0000?style=flat-square&labelColor=000000" />
<img src="https://img.shields.io/badge/MERCADO-Legal_%26_Contable_LATAM-333333?style=flat-square&labelColor=111111" />
<img src="https://img.shields.io/badge/EMBEDDINGS-384D_propios-555555?style=flat-square&labelColor=111111" />
<img src="https://img.shields.io/badge/NEXT.JS-15-ffffff?style=flat-square&logo=nextdotjs&logoColor=000000&labelColor=ffffff" />

**Motor de análisis documental con IA especializado en el mercado legal y contable colombiano.**
Procesa PDFs complejos con búsqueda vectorial semántica — consultas en lenguaje natural sobre múltiples documentos de forma simultánea.

> **Arquitectura de costos:** La vectorización (embeddings 384D) y la búsqueda (pgvector) son **locales** — sin APIs externas en el pipeline de datos. Gemini se usa **únicamente para razonamiento final**, reduciendo el consumo de tokens hasta un 80% frente a un pipeline 100% externo.

**LOGRO CLAVE:** Pipeline RAG propio con embeddings locales y búsqueda nativa en PostgreSQL. Recursive Character Splitting + Re-ranking semántico para textos legales colombianos, logrando respuestas precisas con latencia de milisegundos.

| MÓDULO | IMPLEMENTACIÓN TÉCNICA |
|---|---|
| 🔍 **Búsqueda semántica local** | pgvector en PostgreSQL — similitud coseno en ms · **sin APIs externas en el pipeline de búsqueda** |
| 🤖 **Embeddings propios (costo ≈ $0)** | Vectorización 384D en servidor propio — independencia total de proveedores externos |
| 📄 **Chunking para documentos legales** | Recursive Character Splitting + Re-ranking calibrado para español jurídico colombiano |
| ⚖️ **Orquestación híbrida de LLMs** | Groq (consultas simples, velocidad) ↔ Gemini (razonamiento profundo, multi-doc) · enrutamiento por complejidad |
| 🛡️ **Control de alucinaciones** | Threshold de similitud ajustado + ventana de contexto optimizada — el LLM solo recibe contexto relevante |

```
STACK: Next.js 15 · React 19 · TypeScript · Supabase · PostgreSQL + pgvector · Clerk · LangChain · Groq · Gemini API
```

🔗 **[→ VER DEMO EN VIVO](https://documente.vercel.app/)** &nbsp;|&nbsp; **[→ VER REPOSITORIO](https://github.com/DanielcoderIA/documente)**

### `[03] 📊 VISIÓN 360 — Plataforma de Auditoría Financiera con IA`

<img src="https://img.shields.io/badge/●_STATUS-LIVE-CC0000?style=flat-square&labelColor=000000" />
<img src="https://img.shields.io/badge/DATOS-+10.000_registros_reales-333333?style=flat-square&labelColor=111111" />
<img src="https://img.shields.io/badge/PERFORMANCE-40%25_menos_carga-555555?style=flat-square&labelColor=111111" />
<img src="https://img.shields.io/badge/NEXT.JS-15-ffffff?style=flat-square&logo=nextdotjs&logoColor=000000&labelColor=ffffff" />

**Panel financiero en tiempo real para toma de decisiones estratégicas en LATAM.**
Visualiza y diagnostica más de 10.000 registros financieros reales con análisis automático por IA.

**LOGRO CLAVE:** Reducción del **40% en el tiempo de carga inicial** al migrar de Client-side Fetching a Server Actions con virtualización de datos. Diagnósticos automáticos de flujo de caja mediante Gemini NLP sin intervención humana.

| MÓDULO | IMPLEMENTACIÓN TÉCNICA |
|---|---|
| 🔮 **Diagnósticos automáticos con IA** | Google Gemini NLP — detecta patrones anómalos de flujo de caja y genera alertas para perfiles no técnicos |
| 🌎 **Soporte financiero LATAM** | Normalización de moneda para estándares colombianos, mexicanos y regionales sin pérdida de precisión |
| ⚡ **Migración CSR → Server Actions** | Eliminación de waterfalls de red — 40% menos tiempo de carga vs arquitectura anterior de Client-side Fetching |
| 📈 **Actualizaciones en tiempo real** | Supabase Realtime con WebSocket — sin polling ni refreshes manuales |
| 🚀 **Performance con datos masivos** | Virtualización + lazy loading de +10K puntos de datos sin degradación de la interfaz |

```
STACK: Next.js 15 · TypeScript · React · Supabase · PostgreSQL · Google Gemini API · Tailwind CSS
```
